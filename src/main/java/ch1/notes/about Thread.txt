웹서버(Web Application Server[aka = WAS]) - 스레드 구현되어있음.



구현법
1) 상속
extends Thread
//@Override
public void run(){}
//↓↓↓↓↓↓↓↓run 메소드를 호출하는 방법
A a = new A():
a.start();
Thread th = new A();
th.start();
상속을 했기에 선언부에 상위클래스가 올수있다.

선언부 타입 = 생성부 타입 // 다형성이 아래경우보다 떨어짐.
선언부 타입(추상클래스,인터페이스) != 생성부 타입(구현체 클래스) // 다형성을 누리고 싶다면 이런 구조가 되어야한다.
다형성을 누리고 싶다면, 선언부와 생성부의 타입을 다르게 한다.
유지보수시 코드를 최소한으로 수정하고 기능수정이 가능하게 됨(오타,실수 최소화가능)





2) impliments Runnable 자바는 단일상속만 지원. 그렇기에 보안적으로 인터페이스를 제공
